Anim8 is a graphic language that allows ease of access to basic image creation in the browser.
See the full project on GitHub.

Basics

Anim8 is a very simple and straightforward language. It uses JavaScript to run, so its uses are limited. However, it is still an easy and effective way to display dynamic images in your webpage. While currently its capability is very low, rest assured there will be future updates. Now let's get started with some Anim8.


Environment

Anim8 is an interpreted language, so it can normally only run in a web browser.

In order to run Anim8 in your HTML projects, you must include this code anywhere in the document:

<script src='https://coding-master5.github.io/Anim8/main.js'></script>

After that, you can create a new object of the type "Anim8" and write:

var anim8 = new Anim8;
anim8.parse('//your code here');

Anim8 is run through JavaScript and uses a custom class with a function, parse(), to run its code. Now that your environment is set up, let's get anim8ing!

Your first Anim8 script

Let's write a simple Anim8 script, and then discuss it.

Image init();
new circle '#ff0000', '50%', '50%', '50';
Image end();

Congrats, you wrote your very first Anim8 script! Now let's discuss what it's doing.

Image init();

This keyword is required in all Anim8 scripts. It tells the code to create a new image to append items to. It belongs to the Image class, which is used to initialize and finish the image.

new circle 'color', 'xPos', 'yPos', 'radius'

This keyword creates a new circle with the specified attributes. If none are given, the values default to a generic black circle. This belongs to the new class, which is used to add elements to the image.

Image end();

This keyword is very similar to Image init();, except instead of creating the image, it saves the image as a dataurl and returns it to the function which called it.
