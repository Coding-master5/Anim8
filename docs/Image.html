Class Image

The Image class is the basis of all Anim8 code. It is used to manage images.

Functions

The current functions of this class are init() and end(). Syntax:

Image init();
Image end();

The init() function creates the image upon which to place the image elements. The end() function ends the creation process of the image and returns the image as a dataurl.
