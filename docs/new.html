Method new

The new method is used to insert new elements into an image. It currently has these functions:

new circle 'color(hex)', 'x', 'y', 'radius';
new quad 'color(hex)', 'x', 'y', 'width', 'height', 'rounding';
new line 'color(hex)', 'startX', 'startY', 'endX', 'endY';
new text 'color(hex)', 'x', 'y', 'text';
new image 'image(url)', 'x', 'y';

circle

This creates a circle with customizable color, position, and size.

quad

This creates a rectangle with customizable color, position, size, and border rounding.

line

This creates a line with customizable stroke color and position.

text

This creates a piece of text with customizable color, position, and text.

image

This creates an image with customizable source and position. IMPORTANT: This function has no relation to the Image class. The word 'image' in this context does not refer to Anim8 code.
