Class Image

The Image class is the basis of all Anim8 code. It is used to manage images. Its current functions are:

Image init();
Image end();

init()

This function is used to create a new image. It must be wrote at the beginning of all Anim8 code.

end()

This is used to end the image creation process and return the image data as a dadaurl.
